# Feedra Widget Example - HTML

A simple HTML example demonstrating how to integrate the [Feedra Widget](https://feedrahq.com) into any web page.

## Integration

Add the following script tag before the closing `</body>` tag:

```html
<script
  src="https://widget.feedrahq.com/v1.js"
  data-team-id="YOUR_TEAM_ID"
  async
></script>
```
