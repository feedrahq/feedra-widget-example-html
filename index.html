Integration Instructions

To add the Feedra Widget to your page, include the following script tag before the closing </body> tag: 

<script
  src="https://widget.feedrahq.com/v1.js"
  data-feedra-id="YOUR_PROJECT_ID"
  async
></script>

Note: Replace YOUR_PROJECT_ID with your actual Feedra project ID from your dashboard.
